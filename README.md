# Study Hive

Study Hive is a student-focused study and booking platform designed to make learning more organised, accessible and engaging. Built with HTML, CSS and JavaScript, it features login and signup pages, booking tools, clear instructions and a clean branded interface that helps students manage study sessions with ease and confidence.
